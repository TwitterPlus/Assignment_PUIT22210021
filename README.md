## Tools & Technologies

The following libraries and tools were used to complete these assignments:

**Python 3.9**: Core programming language.

**Pandas & NumPy**: Data manipulation and numerical analysis.

**Matplotlib & Seaborn**: Data visualization and plotting.

**Scikit-Learn**: Machine learning model building and evaluation.

**Jupyter Notebook**: Interactive coding environment.

## How to Run

1. **Clone the repository**:

```bash
git clone [https://github.com/TwitterPlus/Assignment_PUIT22210021.git](https://github.com/TwitterPlus/Assignment_PUIT22210021.git)
```

2. **Install dependencies**:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

3. **Navigate to the desired task folder and run the Jupyter Notebook or Python Project.**
